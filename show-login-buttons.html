3. Manually Fix

If the button above doesn't work, you can manually fix it by opening your browser console (F12) and pasting this code:

const accountSection = document.getElementById('account-section');
if (accountSection) {
  accountSection.innerHTML = `
    <div class="auth-buttons">
      <button class="btn-login" onclick="window.location.href='login.html'">
        <i class="fas fa-sign-in-alt"></i>
        <span>Login</span>
      </button>
      <button class="btn-signup" onclick="window.location.href='signup.html'">
        <i class="fas fa-user-plus"></i>
        <span>Sign Up</span>
      </button>
    </div>
  `;
}
